# 🛡️ CV Web - Valentín Torassa Colombero

Un sitio web CV moderno y profesional para **Cybersecurity & Compliance Analyst** construido con React, TailwindCSS y Framer Motion.

## 🚀 Características

- **Diseño Moderno**: Interfaz minimalista pero impactante con tema oscuro
- **Animaciones Suaves**: Transiciones fluidas con Framer Motion
- **Responsive**: Diseño mobile-first completamente adaptable
- **Profesional**: Enfoque en ciberseguridad y sistemas
- **Interactivo**: Experiencia de usuario tipo dashboard profesional

## 🛠️ Tecnologías

- **React 19** con TypeScript
- **Vite** para desarrollo rápido
- **TailwindCSS** para estilos
- **Framer Motion** para animaciones
- **Lucide React** para íconos
- **Heroicons** para íconos adicionales

## 🎨 Diseño

- **Tema**: Modo oscuro por defecto
- **Colores**: Naranja, azul cian y grises oscuros
- **Tipografía**: Inter (principal) y JetBrains Mono (código)
- **Layout**: Secciones bien definidas con tarjetas y animaciones

## 📱 Secciones

1. **Header** - Información de contacto y navegación
2. **Sobre mí** - Descripción profesional
3. **Experiencia** - Timeline de experiencia laboral
4. **Habilidades** - 4 categorías técnicas principales
5. **Educación** - Formación académica
6. **Certificaciones** - Cursos y certificaciones
7. **Docencia** - Experiencia en enseñanza
8. **Tecnologías** - Badges con Shields.io
9. **Aptitudes** - Habilidades personales
10. **Footer** - Enlaces y mensaje inspirador

## 🚀 Instalación y Uso

```bash
# Clonar el repositorio
git clone <repository-url>
cd mi-cv-web

# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev

# Construir para producción
npm run build

# Preview de producción
npm run preview
```

## 📦 Scripts Disponibles

- `npm run dev` - Servidor de desarrollo
- `npm run build` - Construcción para producción
- `npm run preview` - Preview de la build
- `npm run lint` - Linting del código

## 🎯 Características Destacadas

- **Animaciones al scroll**: Elementos aparecen suavemente al hacer scroll
- **Hover effects**: Interacciones visuales en tarjetas y botones
- **Gradientes**: Efectos visuales modernos con gradientes
- **Badges dinámicos**: Tecnologías con Shields.io
- **Scroll to top**: Botón flotante para volver arriba
- **Background pattern**: Textura sutil de fondo

## 🌐 Deploy

El proyecto está listo para deploy en:
- **Vercel**: `vercel --prod`
- **Netlify**: Arrastrar carpeta `dist` a Netlify
- **GitHub Pages**: Configurar GitHub Actions
